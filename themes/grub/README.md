### Installation

1. Clone repo
1. Move the eldritch folder to /boot/grub/themes
3. Change the theme in /etc/default/grub ( Set GRUB_THEME to '/boot/grub/themes/eldritch/theme.txt')
4. Run sudo grub-mkconfig -o /boot/grub/grub.cfg
