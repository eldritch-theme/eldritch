</p>

### Installation

1. Download your preferred palette's configuration from [themes](themes/) into `~/.config/alacritty`
2. Import your preferred palette's configuration in `alacritty.toml`:
   ```toml
   general.import = [
       "~/.config/alacritty/eldritch-cthulhu.toml"
       # "~/.config/alacritty/eldritch-abyss.toml"
       # "~/.config/alacritty/eldritch-dusk.toml"
   ]
   ```
